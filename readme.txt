Laura RE Site — Standalone Upload

This package is intentionally flat. It contains only:
- index.html
- readme.txt

The logo and hero image are embedded directly inside index.html as data URIs, so you do not need an assets folder.

To use with GitHub / Vercel:
1. Open the zip.
2. Upload index.html and readme.txt to the root of the repository.
3. Commit to main.
4. Vercel will redeploy automatically.

To preview locally:
- Double-click index.html.
